# Gran Lista de Utilidades y Recursos para Diseño y Desarrollo Web #

En este pequeño wiki, se recopilan todos los enlaces hacia los recursos, utilidades, páginas, programas, etc. que puedan ser de utilidad para los diseñadores web y para los desarrolladores web.
Este repositorio de enlaces nace en base a los siguientes artículos de mi blog, [Trazos Web](http://www.trazos-web.com/ "Blogging, Desarrollo y Diseño Web"):

- [74 sitios web útiles para desarrolladores y diseñadores web](http://www.trazos-web.com/2011/05/23/74-sitios-web-utiles-para-desarrolladores-y-disenadores-web/).
- [95 recursos y utilidades para diseño y desarrollo web](http://www.trazos-web.com/2011/10/18/95-recursos-y-utilidades-para-diseno-y-desarrollo-web/). 
- 179 utilidades y recursos para diseño y desarrollo web.

Este repositorio será revisado periódicamente y actualizado contínuamente.

Es de destacar que en este Wiki solo se incluirán los enlaces hacia los recursos y utilidades que considero son los mejores y los más populares debido a la enorme cantidad de recursos que se encuentran en Internet.
